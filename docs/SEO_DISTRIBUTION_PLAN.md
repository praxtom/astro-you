# AstroYou SEO Distribution Plan

## Goal

Turn SEO pages and free tools into a compounding acquisition loop: search visit, useful result, saved profile, first chat, return visit, and paid conversion.

## Weekly Operating Rhythm

1. **Monday: Search Console review**
   - Check indexing status for new pages.
   - Export top queries, pages with impressions but weak CTR, and pages losing clicks.
   - Pick 3 pages to improve based on query evidence.

2. **Tuesday: Content upgrade**
   - Add missing examples, FAQs, internal links, and clearer first-screen tool copy.
   - Keep the tool/action above long explanatory content.
   - Avoid generic filler and repeated keyword blocks.

3. **Wednesday: Free tool improvement**
   - Improve one calculator/tool flow.
   - Track `seo_tool_start`, `seo_tool_complete`, `seo_cta_click`, signup, first chat, and payment attribution.

4. **Thursday: Distribution**
   - Pitch 5 relevant sites or creators.
   - Share one useful tool page in a non-spammy way where it answers a real question.
   - Update one social/video description to link to the most relevant tool.

5. **Friday: Technical check**
   - Run `pnpm test`.
   - Run `pnpm run build` before publishing SEO route or prerender changes.
   - Check sitemap count and confirm new routes exist in `public/sitemap.xml`.

## Backlink Targets

Prioritize relevant, real mentions over volume.

1. Indian lifestyle and culture blogs.
2. Spirituality and yoga creators.
3. Astrology education newsletters.
4. Tool directories that list free calculators.
5. YouTube descriptions for astrology explainer videos.
6. Quora and Reddit answers only when the link genuinely helps the question.
7. Guest posts comparing traditional astrology concepts with practical modern use.

## Page Quality Checklist

Each SEO page should have:

1. A clear H1 matching the search intent.
2. A useful first-screen action or answer.
3. At least three meaningful sections.
4. At least three FAQs.
5. Internal links to the parent cluster and related tools.
6. A conversion path into free Kundali, matching, Panchang, Synthesis, or Consult.
7. Structured data where appropriate.
8. No private app route as canonical.
9. No fake authority, fake testimonials, or guaranteed outcomes.

## Measurement

Track weekly:

1. Indexed pages.
2. Organic impressions.
3. Organic clicks.
4. CTR by page.
5. Tool starts.
6. Tool completions.
7. Saved profiles from SEO.
8. First Synthesis chats from SEO.
9. Consult starts from SEO.
10. Paid conversions from SEO.

## Guardrails

1. Product pages must stay clean and app-like.
2. SEO content belongs below the tool/action, not above it.
3. Do not keyword-stuff.
4. Do not publish unsupported medical, legal, financial, or guaranteed prediction claims.
5. Do not create fake reviews, fake astrologer credentials, or fake social proof.
